중앙선거관리위원회 데이터를 활용한 빅데이터 분석 시스템
======================================================
* #### 프로젝트 주제
  선거종합정보 공공데이터와 선거공약정보 공공데이터를 이용하여 지난 2년간 당선된 후보들의 공약 중 어떤 단어가 공통적으로 많이 언급되었는지에 대해 알아보고자 한다. 

* #### 프로젝트 목적
  당선인의 공약 중 공통적으로 많이 언급된 단어를 파악함으로써 사람들이 후보를 뽑을 때, 어떤 것을 가장 중요시하는지 알 수 있다. 또한, 어떤 공약을 내세웠을 때 당선률이 높을지에 대해서도 알아보고자 한다.

* #### 데이터 수집
  중앙선거관리위원회에서 제공하는 API를 Python을 이용하여 데이터 수집 및 전처리

* #### 데이터 분석
  MapReduce 이용하여 wordcount 수행
